Program ini digunakan untuk menjumlahkan dua buah matriks berukuran sama.
Cara kerjanya:
Pengguna memasukkan jumlah baris dan kolom matriks.
Program meminta pengguna mengisi elemen-elemen matriks pertama dan kedua.
Kedua matriks dijumlahkan elemen per elemen dengan rumus:
hasil[i][j] = matriks1[i][j] + matriks2[i][j].
Hasil penjumlahan ditampilkan dalam bentuk matriks.
